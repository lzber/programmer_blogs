# 双排

不妨调整一下阵型，一排拆为多排，让排与排之间形成高度差，使画面变得高低错落起来。

最简单的，让前一排人蹲，后一排人站着，形成两层高度。

适合人群和场景：稍微比较正式，不挑地形。

# 多排

还是觉得有点呆板？那是因为站得太齐了，两个高度层层内的高差不明显。

那就再调整一下站位，让高低更错落。

比如有人弯着腰，手摆放的位置更亲近自然一点。

相机位置要前下方。

适合人群和场景：关系比较密切，比较熟悉的，不挑地形

Demo
